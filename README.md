# Cohen's Cryptosystem

This repository is an implement of [Cohen's public-key cryptosystem create by Bram Cohen in 1998.](https://web.archive.org/web/20111007060226/http://bramcohen.com/simple_public_key.html) Bram Cohen is best known as [the author of the BitTorrent peer-to-peer file sharing protocol and the Chia Blockchain](https://en.wikipedia.org/wiki/Bram_Cohen).
